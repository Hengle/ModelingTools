ConvertBinary is used to convert binary head or drawdown files from MODFLOW to .csv files.
ConvertBinary can be run from the command line using the form 
ConvertBinary.exe "<input file name>" "<output file name>"
or
ConvertBinary.exe "<input file name>"
With the latter form, the output file name will with the same as the input file name except that the extension will be changed to ".bhd".

ConvertBinary can also be run as a GUI application. In that case, double-click ConvertBinary.exe and a form will be displayed in which you can specify the input file name and optionally the output file name. Then click the Convert button to perform the conversion.

ModelMate: Frequently Asked Questions

    • A.1: The steps for creating instruction file(s) are described on p. 28 of the ModelMate documentation. In ModelMate, use menu item Model|Create Instruction Files For Observations Defined In ModelMuse. To generate the model output file, you'll need to run Ucode (which can be done from ModelMate) so it will run the model. ModelMate and the ModelMate documentation are available free for download here:

      Banta, E.R., 2011, ModelMate--A graphical user interface for model analysis: U.S. Geological Survey Techniques and Methods, book 6, chap. E4

    • A.2: Before attempting to make a parameter-estimation run, ensure that ModelMate is configured correctly to start UCODE_2005 and make a forward model run. Ensure that you have parameters defined. Ensure that template files are defined correctly for each model-input file that UCODE will need to generate to make a model run. Ensure that extraction-instruction files are defined correctly for each model-output file from which simulated values will need to be extracted. Next, make one parameter adjustable and make a sensitivity run. If that is successful, make multiple parameters adjustable, if desired, and make a parameter-estimation run.

    • A.3: ModelMate is a preprocessor for UCODE_2005. Success in running ModelMate is dependent on an adequate understanding of UCODE_2005. Information on use of UCODE_2005 is provided in:

      Poeter, E.P., Hill, M.C., Banta, E.R., Mehl, Steffen, and Christensen, Steen, 2005, UCODE_2005 and six other computer codes for universal sensitivity analysis, calibration, and uncertainty evaluation constructed using the JUPITER API: U.S. Geological Survey Techniques and Methods, book 6, chap. A11.

      Additional information on gradient-based sensitivity analysis and model calibration in general is provided in:

      Hill, M.C., 1998, Methods and guidelines for effective model calibration: U.S. Geological Survey Water-Resources Investigations Report 98-4005, 90 p.

      Hill, M.C., and Tiedeman, C.R., 2007, Effective groundwater model calibration, with analysis of data, sensitivities, predictions and uncertainty: New York, John Wiley & Sons, 464 p. (Requires purchase, but may be available in some libraries.)

    • A.4: GW_Chart provides graphing capabilities for many of the analyses described in Hill (1998) and Hill & Tiedeman (2007). ModelMate has a menu item for starting GW_Chart, but you will need to download and install GW_Chart yourself.

    • A.5: Parameter estimation is a notoriously difficult problem. As one student said: "Great, now there is another way for my model not to converge!"
